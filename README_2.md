# 📓 Notebook App

TODO管理とメモをGitHub Gistに同期するWebアプリです。
ブラウザさえあればどのデバイスからでも同じデータにアクセスできます。

🔗 **アプリURL:** `https://rnakajimanms-blip.github.io/notebook-app/`

---

## ✨ 機能

- ✅ **TODO管理** — タスクの追加・完了・削除・インライン編集
- 📅 **期限設定** — カレンダーUIで完了目標日を設定
- 🔔 **アラート** — 当日〜2週間前まで最大5件の通知タイミングを設定
- 📅 **Googleカレンダー連携** — 期限をGoogleカレンダーに自動登録、アプリを閉じていても通知が届く
- 🗒️ **メモ** — カード形式で自由にメモを作成・編集
- 🔄 **自動同期** — GitHub Gistにリアルタイム同期、どのデバイスからでも同じデータを参照
- 🏷️ **日付管理** — 作成日・更新日・完了日を自動記録

---

## 🚀 セットアップ手順

### 1. GitHubトークンの取得

1. [github.com/settings/tokens](https://github.com/settings/tokens) を開く
2. 「Generate new token (classic)」をクリック
3. 以下を設定：
   - Note: 任意（例: `notebook-app`）
   - Expiration: 任意（`No expiration` 推奨）
   - Scope: **`gist`** にチェック ✅
4. 「Generate token」→ 表示されたトークン（`ghp_xxx...`）をコピーして保存

> ⚠️ トークンは一度しか表示されません。必ずコピーしてください。

---

### 2. アプリにログイン

1. アプリURL（`https://rnakajimanms-blip.github.io/notebook-app/`）をブラウザで開く
2. トークン入力画面が表示される
3. 取得したGitHubトークンを入力して「**はじめる**」をクリック
4. 自動的にGistが作成され、データ同期が開始される

> 💡 トークンはブラウザに保存されるため、**次回以降は入力不要**です。

---

### 3. Googleカレンダー連携（任意）

期限をGoogleカレンダーに自動登録したい場合のみ設定してください。

1. アプリ右上の「**Googleでログイン**」をクリック
2. Googleアカウントを選択して許可
3. 「アプリは確認されていません」と表示された場合：
   - 「詳細」をクリック
   - 「notebook-appに移動（安全ではないページ）」をクリック
4. 「連携中」と表示されれば完了

> 💡 Googleログインは**アプリ起動のたびに必要**です（セキュリティ仕様）。ただしワンクリックで完了します。
>
> 💡 Googleカレンダー連携を使わない日は、ログインせずにアプリを使えます。

---

## 📱 別デバイスでの使い方

1. `https://rnakajimanms-blip.github.io/notebook-app/` をブラウザで開く
2. GitHubトークンを入力して「はじめる」
3. 自動的に既存データが読み込まれる

**データは各自のGitHub Gistに保存されます。** 他のユーザーのデータと混在することはありません。

---

## 🔔 アラート通知について

| 通知方法 | 条件 |
|----------|------|
| **Googleカレンダー通知** | Googleログイン済み＋カレンダー登録ON ← **推奨** |
| **ブラウザ通知** | ブラウザの通知を許可済み＋アプリを開いている |

ブラウザを閉じていてもスマホでも通知を受け取るには、**Googleカレンダー連携**の使用を推奨します。

## 📋 使い方

### TODO

| 操作 | 方法 |
|------|------|
| タスク追加 | 入力欄に入力してEnter |
| 完了 | 左の丸ボタンをクリック |
| 編集 | タスク名を直接クリックして編集 |
| 期限・アラート設定 | 📅ボタンをクリック |
| 削除 | ×ボタンをクリック |

### フィルター

| フィルター | 表示内容 |
|------------|----------|
| ALL | 全タスク |
| 未完了 | 未完了のみ |
| 完了 | 完了済みのみ |
| 期限切れ | 期限を過ぎたタスクのみ |

---

## 🔒 セキュリティについて

- GitHubトークンはHTMLファイルには含まれず、**各自のブラウザにのみ保存**されます
- データはPrivate Gistに保存されるため、**本人のみアクセス可能**です
- このリポジトリをPublicにしてもトークンが漏洩する心配はありません

---

## ⚙️ トークンをリセットしたい場合

アプリ右上の「⚙」ボタンをクリックするとトークン設定をリセットできます。
再度トークンを入力すれば同じデータに再接続できます。
